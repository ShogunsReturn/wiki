# Warehouse

- The warehouse is a <Link to="/docs/game-concepts/buildings#types-of-buildings">Level building</Link>, so we can only increase it one level at a time and its construction has no maximum level.
- The warehouse limits the amount of food, wood, and iron we can store.
- Each additional level improves the amount of each storable resource.
