# Ashihei

## Overview

Ashihei are villagers with basic military training. They fight with self-crafted weapons, making them inexpensive to recruit. Any Ashihei stationed in their own province will work alongside the villagers of that province.

---

- Wood: 0
- Iron: 0

- Population: 1
