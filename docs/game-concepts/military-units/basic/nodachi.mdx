# Nodachi

## Overview

Nodachi units are trained in the use of the nodachi, a large and heavy two-handed sword designed for open-field battles. They are more powerful melee infantry units compared to Bushi.

---

## Key Features

### Cost
- Rice: 80
- Wood: 0
- Iron: 20
- Gold: 25
- Population: 1

### Maintenance
- Meat: 1.6 /h
- Gold: 1.0 /h

### Training duration
- 5 Hours

### Honor requirement
- -100

### Unit type
- <Link to="../#melee-infantry">Melee Infantry</Link>

### Samurai Skills  
They can be affected by the Samurai abilities:  
- <Link to="../../samurais/charisma-skills#exemplary-soldier">Exemplary Soldier</Link>  
- <Link to="../../samurais/charisma-skills#divine-fury">Divine Fury</Link>
