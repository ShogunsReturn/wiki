# Nodachi

## Overview

Nodachi units are trained in the use of the nodachi, a large and heavy two-handed sword designed for open-field battles. They are more powerful melee infantry units compared to Bushi.

---

### Cost
- Rice: 80
- Wood: 0
- Iron: 20
- Gold: 25
- Population: 1

### Maintenance
- Meat: 1.6 /h
- Gold: 1.0 /h
