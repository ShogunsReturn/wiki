# Ashihei With Yari

## Overview

Ashihei with yari are trained in combat using long wooden spears, making them particularly **effective against cavalry units**. Like basic Ashihei, they contribute to the province's workforce when stationed in their own province.

---

## Key Features

### Cost
- Rice: 60
- Wood: 15
- Iron: 0
- Gold: 10
- Population: 1

### Maintenance
- Meat: 1.2 /h
- Gold: 0.6 /h

### Training duration
- 2 Hours

### Honor requirement
- -100

### Unit type
- <Link to="../#melee-infantry">Melee Infantry</Link> specialized in countering cavalry.

### Trait
- <Link to="../#ashihei-units">Ashihei Unit</Link>
- <Link to="../#spear-units">Spear Unit</Link>
